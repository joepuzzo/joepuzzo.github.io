Prefetching And Caching

  • Prefetching: Fetching data into cache before it is requested
  • Caching: storing data away for later use (putting unused MEM back)

Q&A

OS controls HW: Os knows how to interact with hardware components and allocate resources to them

OS controls your program: OS is responsible for allocating some processing power to your program

How does computer handle IO

  • our program has no control outside its memory area
    • Solution: interrupts or traps ( software interrupt )
  • Trap occurs when code executes I/O command
    • OS takes control of the cpu and executes the I/O command on your behalf

How does computer read from disk

  • CPU issues a request to the disk

  • disk sends own interrupt when it completes service
